## Installation

1. Clone the repository to your local machine:
```
git clone https://github.com/bevvee/TodoList-Mobile.git
```

2. Navigate to the project directory:
   
 ```
   cd TodoList-Mobile
```

3. Install the project Node dependencies:
   
```
  npm install
```

4. Start app on Android :
   
```
  npm run android
```
